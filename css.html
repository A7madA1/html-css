CSS Tutorial

Börja använda CSS

Det finns tre sätt att använda CSS på din websida

  • Den första är genom att länka till en extern fil. Detta är den rekommenderade metoden. För att tillampa detta, du måste skriva dina CSS stil i en separat textfil och spara den som en .css fil. Syntaxen för att lägga till stilen i din HTML kod är:
    <link rel="stylesheet" type="text/css" href="style.css"></p>

    Du lägger till<link> taggen i huvudelementet, mellan <head> ... </head> taggar. De två första attributen rel och typ berättar för webbläsaren att detta är en CSS formatmall.Det sista attributet href är där du ange sökvägen till CSS filen som du vill länka till.

  • Den andra metoden för att lägga till CSS stil på en webbsida är att bädda in koden direkt i HTML filen, mellan <head> ... </head>. genom att skriva CSS stilen mellan <style> ... </style>

    Exemple

                                    <head>
                                    <style>
                                          div {
                                          color: blue;
                                          width: 100px;
                                          height: 200px;
                                          } 
                                    </style>
                                </head>

  • Den tredje metoden för att lägga till CSS stil på en webbsida är att använd inline CSS. Inline CSS anges i starttaggen för elementet du vill tillämpa det på, med stilattributet. Varje regel slutar med semikolon (;).

    Exemple

                                    <div style="color:red; background-color:blue;">Some text</div>

Syntax för en CSS regel

selector { property : value; property : value; property : value; }

Om du vill utforma innehållet i en <div> tagg skriver du regeln som exempelen nedanför:

Exemple

                    div {
                        background-color: yellow;
                        font-size: 20px;
                        }

det första ordet div är selector. Det berättar för webbläsaren att uppsättningen regler inne de lockiga parenteserna { ... } gäller alla element med taggen <div>. Inne i de lockiga parenteserna skriver du alla dina deklarationer. Du börjar med att deklarera egenskapen som du vill ange (background-color i den första deklarationen), följt av ett kolon (:). Därefter anger du det värde du vill ha (yellow i förra exempel). Slutligen avslutar du varje deklaration med ett semikolon (;)

Välja ett element

Det finns flera metoder du kan använda för att implementera reglana på elementet

  • - En av de är tagg namnet som före exempel vi har använt div.

  • - Genom att lägga till attribute id till taggen och i vår CSS koden kan du sedan välja respektive id genom att lägga till ett # tecken framför av id namnet. Ett exempel visas nedan:

    Exemple

    Html 
                                    <div id=”aa1”>
                                    text1
                                </div>
                                <div id="aa2">
                                    text2
                                </div
    CSS 
                                    #aa1{
                                font-size: 20px;
                                } 
                                #aa2{
                                font-size: 20px;
                                }

  • Genom att lägga till attribute class till taggen och i vår CSS koden kan du sedan välja respektive id genom att lägga till ett . tecken framför av class namnet. Ett exempel visas nedan:

    Exemple

    Html 
                                    <div class=”class1”>
                                    text1
                                </div>
                                <div class="”class2”">
                                    text2
                                </div
    CSS 
                                    .class1{
                                font-size: 20px;
                                } 
                                .class1{
                                font-size: 20px;
                                }

Okänslighet

För det mesta är CSS väljare och regler skiftlägeskänsliga. Därför kan du antingen skriva små eller stora bokstäver. Båda kommer att fungera lika bra.

Exemple

                        div {
                            Background-color: YELLOW;
                            }
Eller 
                        div {
                            Background-color: yellow;
                            }

Kommentarer

Fvi lägger till kommentarer till vår kod med hjälp av symbolerna /* ... */ .

Exemple

                    /*
                    
                    p {
                        background-color: svart;
                        font-size: 20px;
                        color: white;
                    }
                    */

Positionering

CSS positioneringsegenskapen låter dig placera ett element och ange vilket element ska vara ovanpå vid överlappning.

Det finns fyra metoder för att placera ett element.

  • Statisk positionering

    Statisk betyder inte mycket, det betyder bara att elementen är placerade enligt det normala flödet av sida. Alla HTML element placeras med denna metod som standard. Om du vill ange att statisk positionering ska användas (till exempel för att åsidosätta en annan positioneringsregel för samma element), skriver du:
    position: statisk;

    Exempel

  • Relativ positionering

    Statisk betyder inte mycket, det betyder bara att elementen är placerade enligt det normala flödet av sida. Alla HTML element placeras med denna metod som standard. Om du vill ange att statisk positionering ska användas (till exempel för att åsidosätta en annan positioneringsregel för samma element), skriver du:
    position: relative;

    Exempel

  • Fast positionering

    Som namnet antyder kommer ett element som är placerat med den fasta metoden alltid att stanna på sin tilldelade plats; den rör sig inte ens när sidan rullas. Fast positionering används vanligtvis för att placera sociala delningsknappar på sidan av en webbsida. För att använda fast positionering skriver du :
    position: fixed;

    Exempel

  • Absolut positionering

    När absolut positionering används placeras ett element relativt det första överordnade elementet som har en annan position än statisk. Om inget sådant element hittas placeras det relativt för sidan. Den stannar alltid på samma plats även om sidan rullas.
    position: absolute;

    Exempel

Display och Visibility

CSS har två egenskaper som gör att vi kan ta bort eller dölja ett element. Dessa två egenskaper är Display och Visibility.

  • Display

    Egenskapen CSS display används för att ändra hur ett element visas. Det finns tre vanliga värden: none, inline och block.

    - Det första värdet ”none” får ett element att försvinna. Sidan visas som om elementet inte finns där.

    - Det andra värdet "inline" gör att ett element visas som ett inline element.

    - Det tredje värdet "block" detta element börjar och slutar med en ny rad och sin höjd och bredd kan ändras.

Background

hur man ändrar bakgrundsegenskaperna för en element. Dessa egenskaper inkluderar att ändra dess bakgrundsfärg och bakgrundsbild.

  • Background Color

    För att deklarera bakgrundsfärgen för ett element använder vi egenskapen bakgrundsfärg. I likhet med hur vi anger gränsfärgen för våra CSS rutor kan vi deklarera bakgrundsfärgen för ett element med en av tre notationer: färg namn, rgb notation eller hexadecimal notation.

    Exampel:
    background-color: green;
    background-color: rgb(0, 255, 0);
    background-color: #00FF00;


    Exempel

  • Background Image

    Om du tycker att det är enkelt att bara använda färg som bakgrund för ett element kan du välja att använda en bild. CSS ger oss stor flexibilitet i hur vi vill att bilden ska visas.
    För att använda en bild som bakgrund anger vi URL: en till bilden med hjälp av egenskapen:
    background-image.

Text and Font

Teckensnittsegenskaper

Text och teckensnittsegenskaper i CSS används för att formatera utseendet på ord och text på en webbsida. Teckensnittsegenskapen handlar om hur tecknen ser ut, till exempel om de är "tjocka" eller "tunna", "stora" eller "små" och vilken typsnitt som ska användas.

  • font-family

    Font-family-egenskapen används för att ställa in typsnitt.

    Exampel:
    font-family: "Times New Roman", Times, serif;


    Exempel

  • font-size

    Egenskapen font-size anger textens storlek. Teckensnittsstorlek kan ställas in med pixlar (px), em, procent (%)

  • font-style

    Egenskapen typsnitt används för att ange kursiv text. De två vanliga värdena är normal och italic. Normal visar text utan kursiv medan kursiv visar text i kursiv.

Textegenskaper

I CSS textegenskaper låter dig ange egenskaper som inte är relaterade till textens teckensnitt. Vanliga egenskaper inkluderar textfärg, textjustering, textdekoration, bokstavsavstånd, ordavstånd och radhöjd.

  • color

    Färgegenskapen anger textens färg.

    Exampel:
    color: blue;

  • text-alignment

    Egenskapen textjustering gör att vi kan ange om vi vill att texten ska vara centrerad eller justerad till vänster eller höger eller motiverad. De vanliga värdena är vänster, höger, centrera och motivera.

    Exampel:
    text-align: left;

  • text-decoration

    Den här egenskapen används vanligtvis för att ta bort understrykningar från hyperlänkar. Som standard kommer de flesta webbläsare att visa hyperlänkar i blått, med en understrykning.

    Exampel:
    text-decoration: none;
